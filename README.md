# opentranslate
A free discord translate google extension that automatically translates inbound messages

Allows automatic translation from any supported google translate language to English, Spanish, French, German, and Japanese
More languages coming soon

To Install: Download Plugin.Crx and go to chrome://extensions/ and enable developer mode, then drag and drop the crx file in. 
Alternate method: Download the source code into a zip file, then unpack it to a folder. Once you have all the files raw then go to Extensions and enable developer mode then upload unpacked, and selected the parent directory.
Make sure you do the safety check in the main page and approve the extension if it asks.
Open discord.com or reload the page, it should work automatically. To change the language, pin the extension then click on it.
